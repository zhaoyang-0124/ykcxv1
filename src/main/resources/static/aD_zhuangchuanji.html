📖 装船机作业经验手册

装船机作业经验手册

一、装船作业操作经验

1. 上机前检查经验

  1. 上机前检查行走轨道是否有杂物或者人员。
  2. 检查锚定装置和防风拉锁是否解除，检查确认各夹轮器和电动轮楔是否打开。
  3. 送上控制电后查看“及时报警”“历史报警”是否有故障报警。
  4. 查看各限位、显示器、指示灯、监控、对讲机是否正常。

2. 作业前 操作经验

  1. 与中控核对装船机所要作业的泊位准确无误。
  2. 联系舱口指挥工，按照指令进行动作。
  3. 对好仓位后，大臂上升到停止上限位，抬起安全钩，操作大臂下降到78度出减速限位后，落下安全钩。

3. 作业中 操作经验

  1. 装舱作业时，装船机溜筒与船舶设施保持1米以上的安全距离再作业。
  2. 根据中控要求关注是否有发热煤上船情况。
  3. 查看是否有皮带跑偏撒料情况。

4. 作业后 操作经验

  1. 待料流全部走净后，告知中控，根据指令停止皮带。
  2. 将各旋钮开关打到初始位置。
  3. 按“控制断”按钮断开控制电源。

5. 移仓经验

  1. 移舱时需将移舱旋钮打到“请求”位置。
  2. 通知中控确认信号，待显示屏显示移仓请求确认后，按下“皮带机停机”按钮。
  3. 操作溜筒出舱时，假如船舷太高，需要锁定溜筒出舱的，由专人观察指挥防止溜筒碰撞舱口。
  4. 移仓完成后，待“皮带启动”后旋钮打到移仓“完成”位置。

二、特殊工况操作经验

1. 溜筒无法进仓时经验

对舱时如船舷过高溜筒无法直接进舱需摆动溜筒情况：

  1. 必须俯仰下降到47度以下（溜筒可以锁定角度为47度到15度，锁定范围内溜筒可前摆角度不能超过正10度），操作溜筒锁定。
  2. 同时溜筒进行前摆，不超10度后进行俯仰下降，俯仰和溜筒前摆不可同时操作。

2. 溜筒无法出仓时经验

  1. 出舱时如船舷过高，溜筒无法直接出舱需摆动溜筒时，必须俯仰上升到15度以上（溜筒可以锁定角度为47度到15度，锁定范围内溜筒可前摆角度不能超过正10度），操作溜筒锁定。
  2. 同时溜筒进行前摆不超10度后，进行俯仰上升，俯仰和溜筒前摆不可同时操作。

3. 雾天作业经验

1. 作业前

关注设备状态和视线情况，加强沟通联系。

2. 作业中

  1. 严格控制流量，防止出现皮带打滑导致的重载停机。
  2. 与舱口指挥加强联系，确保大机与船舶保持安全距离。
  3. 如舱口指挥无法看清仓内状况，需提前通知中控停止给料。

3. 作业后

  1. 联系中控，若长时间不作业，升起大臂。
  2. 大机走到锚定位进行防风锚定措施并加强巡视。

4. 湿粘煤作业经验

1. 作业前

与中控确认所装煤种是否存在湿煤，或者所卸列车路程中是否有下雨等情况。

2. 作业中

  1. 如有湿煤上船情况，应及时关闭干雾除尘系统。
  2. 提示中控严格控制流量，防止出现皮带打滑导致的重载停机。
  3. 要时刻关注中心漏斗情况，防止堵料停机。

3. 作业后

  1. 对中心漏斗和大铲进行彻底检查，确保无粘煤，为后续装船作业做好准备工作。
  2. 如有粘煤撒料情况，及时联系清扫人员进行清扫。
  3. 大机走到锚定位进行防风锚定措施并加强巡视。

5. 大风天气作业经验

1. 作业前

  1. 查看现场风速情况。
  2. 要求舱口巡视佩戴护目镜等劳动防护用品。

2. 作业中

  1. 装船机风速仪显示在13.8m/s以内可以作业，超过则停止作业。
  2. 与舱口巡视加强沟通联系，防止大机与船体发生刮碰。
  3. 及时启动干雾除尘系统，防止煤尘飞扬。

3. 作业后

  1. 大机走到锚定位进行防风锚定措施以及加强巡视。
  2. 装船机行走轮前或后需放置铁鞋。

三、故障操作经验

1. 堵煤情况处理经验

  1. 料斗或溜筒堵煤时，须立即停机并报告中控。
  2. 采取安全措施进行排除，严禁重载升臂架、提升溜筒。

2. 走行松缆或紧缆作业经验

走行过程中，“即时报警”栏显示松缆或紧缆故障情况：

  1. 联系装船机巡视到走行电缆卷盘处查看情况。
  2. 装船机巡视进行松缆、紧缆操作，待故障复位后再作业。

3. 溜筒角度超过正负5度、行走限位触发极限位置、臂架伸缩触发极限位置等作业经验

作业过程中设备无动作，“即时报警”栏显示超限位报警情况：

  1. 旋转“极限旁路”按钮不松手。
  2. 同时按下“复位”按钮，再进行相应动作，恢复正常位置。

4. 悬臂皮带跑偏处理经验

作业过程中，“及时报警”栏显示悬皮跑偏情况：

  1. 当悬皮向东侧跑偏时，将尾车推杆拉回。
  2. 当悬皮向西侧跑偏时，将尾车推杆推出。
  3. 提示中控控制料流，观察是否有粘煤情况。

5. 悬臂皮带机无法启动作业经验

  1. 检查控制电源是否送上。
  2. 俯仰工作角度为正14度到负12度，俯仰不在工作角度内将无法启动悬臂皮带机。
  3. 溜筒角度在正负5度以内。

四、其他经验

1. 走行操作作业经验

  1. 走行操作前，确保中控指令到位。
  2. 确保地面上的人员明确知道。
  3. 大车走行时应逐级提升挡位，大车的减速时间是有延迟的，所以当手柄归零时，还会行走一段距离，务必确定好行走距离约1.5米。

2. “就地”操作作业经验

设备就地操作：装船机各机构也可以使用“就地”操作模式在现场进行操作。

  1. 需要在就地操作箱上操作，将该操作箱上的就地选择开关选为“就地”。
  2. 使用就地操作箱上的按钮操作设备（就地操作箱有行走就地操作箱、俯仰就地操作箱、臂架就地操作箱）。

3. 挂安全钩作业经验

  1. 装船作业结束后，在俯仰上升到78度时进入减速区，抬起安全钩。
  2. 直至俯仰到达上限位后（控制屏显示俯仰到达上限位），落下安全钩。
  3. 再将俯仰下降，待控制台上的“进钩”指示灯亮起后完成操作。
  4. 听舱口指挥工指挥，将大机移到锚定位进行锚定，各旋钮开关需打到初始位置后，按“控制断”按钮断开控制电源。

4. 提升环保作业经验

  1. 装船作业时控制物料下落高度，调整臂架角度使物料平缓落舱，避免高速下落产生大量扬尘。

5. 节能降耗作业经验

  1. 作业结束后及时关闭装船机非必要电源、照明等，并有专人进行收尾检查，避免设备长期待机耗电。作业时开启、闲置时关闭，降低照明能耗。
  2. 作业时如发现大铲有粘煤情况。
  3. 可在仓口指挥下摆动抛料板，借用煤流冲击力清理粘煤，提升作业效率。
  4. 装船机司机接到装船指令后再启动设备，皮带采样和移舱等待时，及时停止皮带机运行，减少非必要运转部件，杜绝设备空转浪费。
